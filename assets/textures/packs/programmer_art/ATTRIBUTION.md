## Modifications

Files renamed from upstream ProgrammerArt (legacy MC 1.7-1.12) naming to this project's Alpha-faithful naming convention:

- `log_oak.png` → `log_side.png`
- `log_oak_top.png` → `log_top.png`
- `planks_oak.png` → `planks.png`
- `leaves_oak.png` → `leaves.png` and `leaves_oak_opaque.png` → `leaves_opaque.png`
- `bed_feet_*.png` → `bed_foot_*.png`
- `bookshelf.png` → `bookshelf_side.png`
- `wheat_stage_0..7.png` → `crops_stage_0..7.png`
- `flower_rose.png` → `flower_red.png`
- `flower_dandelion.png` → `flower_yellow.png`
- `pumpkin_face_on.png` → `jack_o_lantern_face.png`
- `pumpkin_face_off.png` → `pumpkin_face.png`
- `rail_normal.png` → `rail.png`
- `rail_normal_turned.png` → `rail_turn.png`
- `slime.png` → `slime_block.png`
- `torch_on.png` → `torch.png`
- `reeds.png` → `sugar_cane.png` (legacy MC name for sugar cane was "reeds")
- `wool_colored_{color}.png` → `wool_{color}.png` (with `silver` → `light_gray`)
- All other files (`stone`, `cobblestone`, `dirt`, `grass_top`, `grass_side`, `sand`, `bedrock`, `brick`, `obsidian`, `cactus_*`, `clay`, `diamond_block`, `door_iron/wood_*`, `furnace_side`, `gold_block`, `ice`, `iron_block`, `jukebox_*`, `ladder`, `mushroom_*`, `pumpkin_side/top`, `snow`, `sponge`, `stone_slab_*`, `tnt_*`) unmodified.

## Tiles not provided by ProgrammerArt

ProgrammerArt targets legacy MC and doesn't ship every Alpha-era tile. BlockAtlas falls back to the `alpha_vanilla` pack for:

- `chest_front/side/top.png`
- `mob_spawner.png`
- `mossy_cobblestone.png`
- `painting_atlas.png`
